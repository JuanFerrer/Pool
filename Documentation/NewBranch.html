How to create a new branch

Making sure you have the right project selected, click the strangely looking icon next to the branch selector.

A small menu should appear. Write the name of the new branch and make sure you select the right branch you want to copy (usually master).

Click "Create new branch". After a moment you should see something like this.

This intends to show the point in time where your branch was created. It should show also the differences between your branch and the comparing one (usually you want to compare with master).
You have now a local branch. Clicking on "Publish" should upload it to GitHub and make it remote.

After that, you can select the branch you're working on and commit changes to that branch.
